Customer Segmentation Dashboard

\*\*An interactive dashboard for customer segmentation analysis using RFM(Recency , Frequency , Monetary) methodology and machine learning clustering techniques.

--Project Overview--
This project analyzes customer purchasing behavior from n online retail dataset and segments customers in
groups for targeted marketing strategies.The analysis uses RFM metrics combined with K-Means clustering to identify customer segments.

R - Recency - Day since last purchase
F - Frequency - Number of purchases
M - Monetary - Total amount spent

--Features--

- Interactive Dashboard: Built with Streamlit for real-time data exploration
- RFM Analysis: Comprehensive Recency, Frequency, and Monetary value analysis
- Customer Segmentation: K-Means clustering to identify distinct customer groups
- Visual Analytics: Multiple interactive charts and 3D visualizations
- Segment Insights: Actionable recommendations for each customer segment

--Dataset--

- Source : Online retail dataset (https://www.kaggle.com/datasets/vijayuv/onlineretail)
- Size: 541,000+ transactions from 4,338 customers
- Period: December 2010 - December 2011
- Region: UK-based online retail company

--Technologies Used--

- Python 3.8+
- Streamlit: Interactive web dashboard
- Pandas: Data manipulation and analysis
- Scikit-learn: K-Means clustering
- Plotly: Interactive visualizations
